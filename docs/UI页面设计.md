# UI 页面设计

Streamlit 工作台包含 8 个页面：

1. 知识库管理：上传制度文件，查看条款 chunk 和入库状态。
2. 报销审核：上传报销单、发票、审批流程，触发智能审核。
3. 合同审核：扫描合同付款风险。
4. 发票识别：单独测试 OCR adapter。
5. Agent 执行链路：展示任务识别、解析、检索、规则判断、校验、报告。
6. 自动化评测：运行测试集并展示指标。
7. 数据飞轮：提交反馈，挖掘 hard case，生成训练数据。
8. 审核报告记录：预览和下载历史 Markdown 报告。

页面层只负责上传、触发和展示，业务判断全部放在后端模块，避免 UI 和核心逻辑耦合。
