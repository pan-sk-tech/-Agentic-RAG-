# 评测体系设计

评测集位于：

```text
fin_compliance/eval/test_cases.jsonl
```

每条 case 包含输入报销单、预期结论、预期风险类型和预期证据条款。

## 指标

- `conclusion_accuracy`：结论是否正确。
- `rule_accuracy`：规则判断是否正确。
- `risk_type_recall`：风险类型召回率。
- `risk_type_precision`：风险类型精确率。
- `evidence_recall`：制度证据召回率。
- `citation_accuracy`：风险项是否绑定证据。
- `faithfulness`：报告是否忠实于检索证据。
- `action_success_rate`：工作流是否完成。
- `hallucination_rate`：是否存在无证据引用。

运行：

```powershell
python fin_compliance_cli.py eval
```
