# 项目介绍

FinCompliance-Agent 是一个企业财务合规审核工作台，目标不是做普通知识库问答，而是完成“材料上传 -> 制度检索 -> 规则判断 -> 证据校验 -> 报告生成 -> 评测反馈”的业务闭环。

## 业务场景

- 差旅报销审核：住宿费、餐饮费、交通标准、发票日期、发票抬头、审批链。
- 发票识别：从 OCR 文本、图片或 sidecar JSON 中抽取金额、日期、购买方、销售方。
- 合同审核：识别预付款比例过高、缺少验收条款、自动续约等风险。
- 审核报告：输出 Markdown 和 JSON，包含结论、风险类型、制度依据和整改建议。

## 技术路线

- LangGraph：编排 Agentic Workflow。
- RAG：从制度条款中检索证据。
- Rule Engine：执行确定性财务规则。
- Verifier：校验证据链，降低幻觉。
- Eval：用固定测试集衡量质量。
- Data Flywheel：沉淀人工反馈、hard case、synthetic data、SFT/DPO 数据。
